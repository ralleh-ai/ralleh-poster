# RALLEH Poster Design Standards

This document establishes guidelines to ensure all generated event posters remain high-quality, distinctive, and completely free of "AI slop" (generic, overly busy, highly polished plastic rendering styles).

## Anti-Slop Guidelines
1. **Deliberate Typography**: Do not let the image model hallucinate raw text. Prefer generating beautiful background visual plates first, or use models with high typographic fidelity while providing strict layout constraints.
2. **Minimalist Composition**: Restraint over clutter. Focus on a singular compelling visual metaphor.
3. **Restricted Color Palettes**: Limit visual output to 3-4 dominant colors. High-contrast or carefully graded duotones often produce premium outcomes.
4. **Art Style References**: Emulate physical artistic print styles:
   - Screen printing (silkscreen texture, slight registration offsets)
   - Risograph (vibrant spot-colors, grain, overlay transparency)
   - Linocut / Woodblock (strong, bold physical cuts)
   - Swiss Design (grid-driven, strong typography, geometric focus)
